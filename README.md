# CollabCode-Backend

## Project Title
CollabCode Backend

## Description
This project provides the backend API for CollabCode, enabling users to collaborate on coding projects in real time. The backend is built with a focus on scalability, security, and ease of use.

## Features
- User authentication and authorization
- Real-time collaboration
- Code versioning
- Notifications for code changes

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Adityaranaa01/CollabCode-Backend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CollabCode-Backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
- API documentation can be found at `/api/docs`.
- For real-time collaboration, ensure you have WebSocket support.
